# История изменений

Все значимые изменения в проекте Incline3D документируются в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
проект придерживается [Семантического версионирования](https://semver.org/lang/ru/).

## [Не выпущено]

### Планируется в версии 0.1.0

#### Добавлено
- Базовая архитектура приложения (слои: Model, Core, IO, Rendering, UI, App)
- Структуры данных для инклинометрии:
  - `IntervalData` — исходные данные интервала
  - `MeasurementPoint` — точка замера
  - `WellResult` — результаты обработки
  - `ProcessedPoint` — обработанная точка
  - `ProjectPoint` — проектная точка
  - `ShotPoint` — пункт возбуждения
  - `Project` — проект (набор скважин)
- Алгоритмы расчёта траектории:
  - Average Angle (усреднение углов)
  - Balanced Tangential (балансный тангенциальный)
  - Minimum Curvature (минимальная кривизна)
  - Ring Arc (кольцевые дуги)
- Расчёт интенсивности искривления (dogleg severity)
- Расчёт погрешностей координат
- Обработка вертикальных участков
- Импорт данных:
  - CSV с настраиваемым маппингом полей
  - LAS 2.0
  - ZAK
- Экспорт данных:
  - CSV
  - LAS 2.0
- Собственный формат проекта (*.inclproj)
- Графический интерфейс GTK4:
  - Главное окно с меню и панелью инструментов
  - Таблица скважин
  - Таблица проектных точек
  - Таблица пунктов возбуждения
  - 3D аксонометрия (OpenGL)
  - План (Cairo)
  - Вертикальная проекция (Cairo)
- Экспорт изображений (PNG, BMP)
- Копирование изображений в буфер обмена
